## 🛠️ Tecnologías Aplicadas en esta Web
* **HTML5:** Estructura limpia y semántica.
* **CSS3:** Estilos totalmente personalizados (sin frameworks como Bootstrap) para lograr el look pixelado, animaciones de naves y efectos CRT de televisión vieja.
* **JavaScript:** Para el menú hamburguesa y el efecto de máquina de escribir en el texto de presentación.
* **Responsive Design:** Adaptado para que se vea cómodo y sin fallas tanto en computadoras de escritorio como en celulares y tablets.
